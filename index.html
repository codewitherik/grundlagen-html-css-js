HTML, CSS & JS

HTML – Die Struktur des Webs

HTML (HyperText Markup Language) ist die Standardauszeichnungssprache für Webseiten. Sie dient dazu, die Struktur und den Inhalt einer Webseite zu definieren.

CSS – Das Styling für Webseiten

CSS (Cascading Style Sheets) steuert das Design und Layout von HTML-Dokumenten. Es ermöglicht Farben, Schriftarten und Abstände zu definieren.

JavaScript – Interaktivität im Web

JavaScript ist eine Skriptsprache, die Webseiten interaktiv macht. Es ermöglicht dynamische Funktionen, Animationen und Benutzerinteraktionen.
